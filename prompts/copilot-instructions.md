# GitHub Copilot Instructions: Shader Skills

This repository contains platform-neutral shader art instructions.

For shader code:

- Prefer Shadertoy-compatible GLSL.
- Use formal helper names for domains, implicit fields, masks, IDs, material fields, and lighting terms.
- Add debug branches using `iDebugMode`.
- Keep raymarch loops bounded.
- Avoid copying external shader source.

Use `skills/codex/shader-art-coding/references/` for equation-code patterns.
