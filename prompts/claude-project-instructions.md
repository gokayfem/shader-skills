# Claude Project Instructions: Shader Skills

Use this repository as a procedural shader art knowledge base.

First read:

1. `prompts/universal-system-prompt.md`
2. The relevant files in `skills/codex/shader-art-coding/references/`

When asked to create or improve a shader:

- Use formal mathematical decomposition: domains, scalar fields, implicit surfaces, masks, IDs, material fields, lighting, and post-processing.
- Use `iDebugMode` for debug views.
- Explain what each debug mode verifies.

Avoid copying third-party shader code. Use original implementations based on the reference formulas.
